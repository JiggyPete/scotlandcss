An A to Z of CSS

This talk will cover a large number of interesting tools and techniques for CSS, in fact, one for each letter of the alphabet! You’ll learn interesting layout modes such as Grid Layout and recap on exactly what the Box Model is. For each tool, technique or property, we’ll see examples & use cases.

Shedding light on the shadow dom

We have many problems that exist in current CSS architecture e.g encapsulation, dead code & specificity conflicts. I will explain how the shadow dom spec can help to solve them by using the pseduo selectors ::host, ::shadow & /deep/. I will also cover how to theme & style native elements.
